Sentiment Analysis and Topic Extraction
This contains code to perform sentiment analysis and topic extraction on a dataset of customer feedback. The code is written in Python and uses the Gensim library for topic modeling and NLTK for text preprocessing.

Requirements
- Python 3.x
- Gensim
- NLTK

Instructions
1.Download Dependencies:
   Ensure you have the required dependencies installed
2.Prepare the Data:
   - Place the CSV file containing customer feedback in the root directory with the filename 'input.csv'
3.Install Required Libraries.
4.Run the code for performing classification task.
Now, new file will be saved as output.csv.

## Word Segmentation
5.Import the libraries and load the 'output.csv' data.
6.Initialize the SentimentIntensityAnalyzer and run extract_words function.
7.New file will be saved as wordsnew.csv, it contains corresponding words.

## WordCloud
8. Plot the word cloud for each category and save it as png.
